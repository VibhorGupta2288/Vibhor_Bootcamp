# Eco-Transit Command Center: Raipur-Bhilai Dust Mitigation

![Dashboard Preview](https://img.shields.io/badge/Status-Active-success) ![Tech Stack](https://img.shields.io/badge/Tech-HTML%20%7C%20CSS%20%7C%20JS-blue)

A real-time, interactive dashboard designed to visualize the severe dust pollution caused by heavy vehicular traffic in the Raipur-Bhilai industrial corridor, and to simulate a highly effective, technology-driven solution.

## 🚨 The Problem: Fugitive Dust Crisis
The Raipur-Bhilai region is a massive industrial and steel hub. Every day, over 15,000 heavy trucks transport raw materials like coal and iron ore. Because many of these trucks use open beds or ineffective manual tarps, wind shear causes immense amounts of fugitive dust (up to 37.5 tonnes daily) to escape into the atmosphere. This significantly spikes the local AQI, causing severe environmental and health hazards.

## 🛡️ The Proposed Solution
This project proposes a two-part mitigation architecture to bring dust emissions to near-zero:
1. **Material Innovation:** PVC-Coated High-Tenacity Polyester Tarpaulins combined with Polyurethane (PU) sealing lips for maximum durability and particle containment.
2. **IoT Automation:** A motorized retractable tarping system equipped with rear optical dust sensors. 
3. *(Optional)* **Electrostatic Suppression:** Tailgate ionization bars that clump and ground escaping PM2.5 particles.

## ✨ Dashboard Features
*   **Real-time AQI Simulation:** Dynamic, animated counters simulating fluctuating traffic-induced AQI levels.
*   **Interactive System Engagement:** A "Cyber Button" that allows users to toggle the Tarping System on and off, dynamically updating all charts, metrics, and visual states in real-time to demonstrate the massive 95% reduction in dust.
*   **Premium UI/UX:** Built with a high-tech "command center" aesthetic featuring glassmorphism, glowing neon accents, staggered load animations, and ambient cursor lighting.
*   **Data Visualization:** Interactive Source Apportionment (Doughnut) and Mitigation Efficiency (Bar) charts powered by Chart.js.

## 🚀 How to Run Locally

This dashboard is built entirely with pure client-side web technologies and requires absolutely **no build tools, no Node.js, and no server** to run!

1. Clone or download this repository to your local machine.
2. Navigate to the project folder.
3. Simply double-click the `index.html` file to open it in your preferred modern web browser (Chrome, Edge, Firefox, Safari).
